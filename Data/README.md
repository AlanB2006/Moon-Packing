# Archived data

The data in this directory were supplied with the revised manuscript package.
They are retained byte-for-byte and are not rewritten by the plotting scripts.

- `sweeps/`: results used for the lifetime and maximum-eccentricity panels.
- `eccentricity_maps/`: results used for the \(\log_{10}\Delta e\) maps.
- `Simulation_Archives/`: REBOUND `*.bin` SimulationArchive files used by the legacy `Fig02.py` workflow. The original author bundle used the path `Data/Simulation_Archives/`; filenames should not be changed.
- `generated/`: destination for newly run validation, grid, and time-series products.

See `MANIFEST.csv`, `checksums.sha256`, and `docs/DATA_DICTIONARY.md`.
