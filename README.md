# Tidal evolution of packed moons around an earth mass planet

This repository contains processed data and Python scripts used in the article:

"Tidal evolution of packed-moons around an earth mass planet"
(Briseno, Quarles, 2026)

The paper investigates whether packed moon systems of equal mass can stably exist around an earth mass planet, and how tidal evolution driven by the host planet & star influences long-term moon survival.
